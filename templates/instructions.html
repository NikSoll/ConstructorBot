📚 Инструкция по настройке бота

Пошаговое руководство для быстрого старта

1 Создание бота в ВКонтакте

📌 Шаг 1.1: Создайте сообщество

Для начала вам нужно создать сообщество ВКонтакте, которое будет вашим ботом:

  1. Перейдите на vk.com/groups
  2. Нажмите кнопку «Создать сообщество»
  3. Выбирайте название и тематику под ваши интересы
  4. Нажмите «Создать сообщество»
Создание нового сообщества

🔑 Шаг 1.2: Получите токен сообщества

  1. Зайдите в управление сообществом (кнопка «Управление»)
  2. В левом меню выберите «Работа с API»
  3. Нажмите «Создать ключ»
  4. Выберите все разрешения (галочки)
  5. Нажмите «Создать» и скопируйте полученный токен
  6. Перейдите во вкладку Long Poll API
  7. Переключите с Выключено на Включено
  8. Перейдите во вкладку Типы событий
  9. Пометьте все пункты групп Сообщения и Фотографии
Создание ключа доступа API
💡 Совет: Токен выглядит как vk1.a.1234567890abcdef.... Сохраните его в надежном месте — он нужен для работы бота.

🆔 Шаг 1.3: Получите ID группы

ID группы — это цифры из адресной строки вашего сообщества:

  • Откройте ваше сообщество в браузере
  • Посмотрите на адресную строку: https://vk.com/club123456789
  • Цифры 123456789 — это ID вашей группы
Как найти ID группы (цифры после club)
⚠️ Важно: Если в адресной строке написано название, а не club+цифры — нажмите на три точки под аватаром и выберите «Скопировать ссылку». В ссылке будут цифры.

🤖 Шаг 1.4: Включите возможности бота

Чтобы бот мог отвечать на сообщения, нужно включить функцию «Возможности бота»:

  1. В управлении сообществом перейдите в раздел «Сообщения» потом «Настройки для бота»
  2. Включите переключатель «Возможности бота» (должен стать синим)
⚠️ Важно: Без включённых «Возможностей бота» бот не сможет отправлять сообщения!

2 Настройка Google Sheets

Для сохранения данных вашего бота (записи, заказы, результаты опросов) используются Google Таблицы.

📊 Шаг 2.1: Какие листы нужны для каждого типа бота

💅 Запись (make)

Нужен лист: Записи

Сохраняет: имя, телефон, услугу, мастера, дату, время

🛍️ Магазин (shop)

Нужен лист: Заказы

Сохраняет: имя, телефон, адрес, корзину, сумму

🎯 Квиз (quiz)

Нужен лист: Результаты квизов

Сохраняет: результаты тестов, баллы

📋 Опросник (survey)

Нужен лист: Результаты опросов

Сохраняет: ответы на вопросы (в JSON формате)

📢 Рассыльщик (mailer)

Листы НЕ НУЖНЫ! Данные хранятся в самом боте

Подписки, рассылки и история — встроенное хранилище

📝 Шаг 2.2: Создание Google таблицы

  1. Зайдите в Google Sheets
  2. Создайте новую таблицу (кнопка «+»)
  3. Назовите таблицу, например: Данные бота
  4. Откройте доступ к таблице во вкладке Настройка доступа
  5. Создайте нужные листы (вкладки внизу) по инструкции выше
  6. Скопируйте ссылку на таблицу из адресной строки
Создание новой таблицы и листов

🔐 Шаг 2.3: Настройка доступа для бота

Чтобы бот мог записывать данные, нужно открыть доступ к таблице:

  1. В правом верхнем углу таблицы нажмите «Настройки доступа»
  2. Нажмите «Настроить доступ»
  3. Выберите «Все, у кого есть ссылка» → «Редактор»
  4. Скопируйте ссылку и вставьте в форму создания бота
Открываем доступ по ссылке (достаточно «Редактор»)
💡 Важно: Ссылка должна выглядеть так: https://docs.google.com/spreadsheets/d/1AbCdEfGhIjKlMnOpQrStUvWxYz/edit

🔐 Шаг 2.4: Добавьте сервисный аккаунт в доступ

Если бот не может сохранить данные в таблицу, возможно, нужно добавить сервисный аккаунт вручную:

  1. Откройте вашу Google Таблицу
  2. Нажмите «Настройки доступа»
  3. В поле для ввода email добавьте адрес сервисного аккаунта:
    marketplace-bot@digitalmarketplace-492113.iam.gserviceaccount.com
  4. Выберите роль «Редактор»
  5. Нажмите «Отправить»

3 Как узнать свой ID

ID нужен, чтобы вы (как администратор) получали уведомления от бота.

📱 Для VK:

  1. Перейдите в свой профиль ВКонтакте
  2. Посмотрите на адресную строку: https://vk.com/id123456789
  3. Цифры после /id — ваш ID
Как найти свой VK ID (цифры после /id)
⚠️ Внимание: Если в адресной строке написано https://vk.com/ваше_имя — ваш ID скрыт. Откройте вкладку "Редактировать профиль", далее "Изменить в VK ID", под вашими ИмяФамилия будет ID.

4 Запуск бота

🚀 После создания бота:

Вы получаете ZIP-архив с тремя файлами:

  • bot.exe — сам бот (с графическим интерфейсом)
  • config.json — настройки вашего бота
  • creds_*.json — ключ для доступа к Google Sheets (если вы его указали)

📁 Запуск бота:

  1. Распакуйте ZIP-архив в отдельную папку (не запускайте из временной папки)
  2. Убедитесь, что все три файла лежат в одной папке
  3. Дважды кликните по bot.exe
  4. Откроется графическое окно с кнопками «Запустить бота» и «Остановить бота»
  5. Нажмите «Запустить бота» — бот начнёт работу
💡 Совет: Если при запуске открывается чёрное окно (консоль) — это нормально для первой версии. В следующих обновлениях мы её уберём.

⚠️ Возможные ошибки при запуске .exe:

  • «Не найден config.json» — убедитесь, что файл лежит в той же папке, что и bot.exe
  • Ошибка 403 при подключении к Google Sheets — добавьте сервисный аккаунт в доступ к таблице
  • Бот не отвечает в VK — проверьте, что включены «Возможности бота» в настройках сообщества

💬 Первый запуск:

После запуска бота отправьте команду /start в диалог с вашим ботом.

Если вы администратор (ID совпадает), вы увидите админ-панель с кнопками:

  • 📊 Статистика
  • 📝 Управление рассылками
  • ✉️ Отправить сообщение
После запуска отправьте боту команду /start
💡 Совет: При первом запуске рассыльщик автоматически создаст в Google Sheets листы "Подписчики", "Рассылки" и "История рассылок", если их нет.

5 Часто задаваемые вопросы

❓ Бот не отвечает

Проверьте, что:

  • Токен сообщества правильный и активный
  • В сообществе включены «Сообщения сообщества»
  • В настройках API выбраны все разрешения

❓ Данные не сохраняются в Google Sheets

Проверьте:

  • Правильно ли скопирована ссылка на таблицу
  • Открыт ли доступ «Все, у кого есть ссылка» → «Редактор»
  • Есть ли в таблице нужные листы (названия должны совпадать)

❓ Не приходят уведомления администратору

Убедитесь, что:

  • Вы указали правильный ID администратора
  • У вас открыт диалог с ботом
  • Бот не заблокирован вами
